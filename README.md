# Information_Security_assignment
# Simple File Encryption and Decryption Tool
Information Security - Course Project 4

## Requirements
pip install cryptography

## Usage

### Encrypt
python3 secure_file_tool.py encrypt --input sample_plaintext.txt --output sample_encrypted.txt --key "YourKey"

### Decrypt
python3 secure_file_tool.py decrypt --input sample_encrypted.txt --output recovered.txt --key "YourKey"

### GUI
python3 secure_file_tool.py gui
